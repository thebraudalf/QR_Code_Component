### What I learned
Here I learned how to use media query properly.

code snippet, see below:

```css
@media  screen and (max-width : 400px) {
 
    *{
        margin: 0;
        padding: 0;
    }

    .bg{
        width: 320px;
        height: 100vh;
        margin: 0 10px;
        padding: 6px;
    }
    
    .component img{
        width: 310px;
        height: 50vh;
    }
}
```
